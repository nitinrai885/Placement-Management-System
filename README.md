# Placement Management System

This is a role-based web application developed using Django that connects students and companies through a simple placement portal.

Students can register, complete their profile, and apply for jobs, while companies can post job opportunities and manage applicants.

---

## 🚀 Features

- Student & Company Role-Based Login
- Profile Management
- Job Posting by Companies
- Job Application by Students
- Application Tracking System
- Select / Reject Candidate Functionality
- Secure Authentication
- Responsive UI using Bootstrap

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- Bootstrap
- Django REST Framework

---

## 📂 Project Structure

- User App – Handles registration, login, and profile management  
- Jobs App – Manages job posting and listing  
- Applications App – Tracks job applications and status  
- API App – Provides backend API endpoints  

---

## ⚙️ How to Run the Project

Follow these steps to run the project locally:

### 1. Clone the Repository
git clone <your-github-repo-link>  
cd placement-portal  

### 2. Install Dependencies
pip install -r requirements.txt  

### 3. Run Migrations
python manage.py migrate  

### 4. Start the Server
python manage.py runserver  

Open browser and go to:
http://127.0.0.1:8000/
