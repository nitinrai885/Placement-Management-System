# Placement Management System

This is a role-based web application developed using Django that connects students and companies through a placement portal.

Students can register, complete their profile, upload resumes, and apply for jobs, while companies can post jobs and manage applicants.

---

## 🚀 Features

- Student & Company Role-Based Login
- Profile Management System
- Resume Upload & Download
- Job Posting by Companies
- Job Application by Students
- Application Tracking System
- Select / Reject Candidate Functionality
- "Already Applied" Status for Jobs
- Secure Authentication
- Responsive UI using Bootstrap

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- Bootstrap
- Django REST Framework

---

## 📂 Project Structure

- User App – Handles registration, login, and profile management  
- Jobs App – Manages job posting and listing  
- Applications App – Tracks job applications and status  
- API App – Provides backend API endpoints  

---

## ⚙️ How to Run the Project

### 1. Clone the Repository
git clone <your-github-repo-link>  
cd placement-portal  

### 2. Install Dependencies
pip install -r requirements.txt  

### 3. Run Migrations
python manage.py migrate  

### 4. Start the Server
python manage.py runserver  

Open browser and go to:
http://127.0.0.1:8000/
